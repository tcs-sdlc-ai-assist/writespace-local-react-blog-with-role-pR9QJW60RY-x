# writespace

A modern blogging platform built with **React 18**, **Vite**, and **Tailwind CSS**.

---

## 🚀 Overview

**writespace** is a full-featured blogging platform that allows users to write, read, and manage blog posts. It supports user authentication, role-based access (admin/user), and a clean, responsive UI powered by Tailwind CSS.

---

## 🛠️ Tech Stack

- [React 18](https://react.dev/)
- [Vite](https://vitejs.dev/)
- [Tailwind CSS](https://tailwindcss.com/)
- [PropTypes](https://www.npmjs.com/package/prop-types)
- [React Router](https://reactrouter.com/)
- [Vercel](https://vercel.com/) (deployment)

## ⚡️ Getting Started

### 1. Clone the repository

```bash
git clone https://github.com/your-username/writespace.git
cd writespace
```

### 2. Install dependencies

```bash
npm install
```

### 3. Start the development server

```bash
npm run dev
```

Open [http://localhost:5173](http://localhost:5173) to view the app in your browser.

### 4. Build for production

```bash
npm run build
```

## 📝 Usage

- Register a new account or log in.
- Write, edit, and delete your own blog posts.
- Admin users can manage users and view platform statistics.
